About this website



This website is devoted to Conway's Game of Life.
On this website you will find in-depth articles
about certain selected topics. If you do not
understand a term used on this website, concise
definitions can be found by browsing Stephen Silver's
Life Lexicon:

http://www.argentum.freeserve.co.uk/lex.htm

On this website there are lots of patterns, both
discovered and engineered. In your browser, you can
simply click on one of these pictures to download
the RLE pattern. RLE is the bog-standard format for
life patterns, since it is compressed and doesn't
contain any control characters.
